Default model

For every order, Autonomous Quotation calculates the predicted cost, forming the basis of our quote calculation. The actions in the rule manager adjust this cost, often by adding additional profit margin on top. Here you can choose between our machine learning algorithm, a fixed cost per distance or an external price API based model. You can add exceptions below, for example when you have different costs on a specific route.
