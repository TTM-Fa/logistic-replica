Advanced ruleset editor

The advanced ruleset editor offers a more flexible complex interface for creating and editing rulesets in Autonomous Quotation. While the standard ruleset editor is used by default, pro-users can convert their rulesets to an advanced editor for more granular control over their quoting strategies. However, once changes made in the advanced editor are saved, the ruleset cannot be converted back to the standard version.

Key features of the advanced editor

  • Nested rules for complex logic
  • Detailed control over pricing strategies

When to use the advanced editor

Consider using the advanced ruleset editor when:

  • You need to create highly specific or complex quoting rules
  • Your quoting strategy requires nested conditions

Accessing the advanced editor

  1. Navigate to the Rule manager in Autonomous Quotation
  2. When editing an existing ruleset, look for the Switch to advanced mode option

Note: switching to the advanced mode is irreversible for the given ruleset

Key differences with the standard ruleset editor

  • More flexible rule creation, including nested and generic rules
  • Greater complexity, requiring more time and expertise to manage
  • Access to legacy features not available in the simplified view

While the advanced ruleset editor offers more control, it also requires a deeper understanding of Autonomous Quotation's capabilities. New users are encouraged to start with the standard ruleset editor before exploring the advanced options. For additional support, don't hesitate to reach out to customer support.
